### ğŸš€ Getting Started

To run any of the projects in this repository, you will need to have the correct development environment set up.

#### Prerequisites

*   **Microsoft Visual Studio**: It is recommended to use [Visual Studio 2022](https://visualstudio.microsoft.com/).
*   **.NET SDK**: Ensure you have the appropriate .NET SDK installed. During the Visual Studio installation, select the **".NET desktop development"** workload.

#### Installation & Running

1.  **Clone the repository:**
    ```sh
    git clone https://github.com/[your-username]/OOP2-FALL-2025-26.git
    cd OOP2-FALL-2025-26
    ```

2.  **Navigate to a project directory:**
    ```sh
    # Example for a lab task
    cd Lab/LabTask1/

    # Or for the final project
    cd Project/
    ```

3.  **Open the Solution in Visual Studio:**
    *   Find the `.sln` file (e.g., `labTask1.sln`) in the project's root directory.
    *   Double-click the file to open it in Visual Studio.

4.  **Build and Run the Project:**
    *   Once the solution is loaded, ensure the correct project is set as the "Startup Project" by right-clicking it in the Solution Explorer and selecting `Set as Startup Project`.
    *   Press `F5` or click the green "Start" button in the toolbar to build and run the application.
